About

Hi, I'm a freelance game developer! I currently work as a full-stack web developer for Holley Performance Products. I also teach kids how to code at Awesome Inc U. When I'm not doing any of those things, I'm probably working on my own games.

My Bachelor's in Computer Science was for game design. I've been educated and trained for every stage of the development process, including design, prototyping, programming, presenting, and marketing. Designing and writing games has been my favorite passion for as long as I can remember.

I'm always looking for cool projects, so don't hesitate to reach out if you're interested! If you happen to find yourself in Lexington, KY, be sure to check out RunJumpDev, a nonprofit dedicated to growing and cultivating the local game development community in Kentucky.
